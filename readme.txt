To use this python project you need:

-Install python, look at this youtube video: https://www.youtube.com/watch?v=e70ykVBazAg

-Install ffmpeg:
 in your CMD or terminal run this command: winget install Gyan.FFmpeg
 to check if it's done, type ffmpeg --version

-Install python libraries:
 also in yout CMD run this command: pip install colorama yt-dlp
